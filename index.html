LLM Server Controller

Кроссплатформенное настольное приложение для удобного управления локальным сервером llama-server.exe. Запускайте и настраивайте GGUF модели без терминала.

📥 Скачать для Windows (.exe) ☕ Поддержать разработку

Основные возможности

🕹 Управление бэкендом

Запуск, остановка и перезапуск сервера с визуальным контролем статуса, анимацией загрузки и звуковыми оповещениями.

📦 Авто-установка

Прямое скачивание 9 различных сборок llama.cpp (Vulkan, CUDA, CPU и др.) прямо из меню. Автоматическая загрузка необходимых DLL.

⚙️ 12 категорий настроек

Тонкая конфигурация: контекст, GPU-слои, Flash Attention, размеры batch, параметры генерации, настройки MoE и эмбеддингов.

Продуманный интерфейс

Главное окно разделено на три функциональные зоны для максимальной продуктивности:

  • Панель путей: Быстрый выбор исполняемого файла, GGUF-модели и кнопка мгновенной загрузки бэкенда.
  • Двухпанельный менеджер параметров: Слева удобный список категорий, справа — аккуратная сетка из 4 колонок со всеми полями ввода.
  • Окно логов: Интерактивный вывод потоков сервера с возможностью очистки, сохранения в файл или копирования в буфер.

Как это работает?

Шаг 1

Скачайте бэкенд через Файл → Установить llama.cpp или укажите путь к своему файлу.

Шаг 2

Выберите вашу любимую ИИ-модель в формате .gguf через файловый диалог.

Шаг 3

Отрегулируйте параметры (например, количество GPU-слоёв для ускорения) под ваше железо.

Шаг 4

Нажмите Запустить. Веб-интерфейс автоматически откроется на порту 18080!

Системные требования

  • ✔ Windows: Полная поддержка (включая встроенный установщик бэкендов).
  • ℹ Linux / macOS: Ограниченная поддержка (установка бэкенда силами пользователя, запуск сервера работает отлично).
  • ✔ Python: Версии 3.x для сборки из исходного кода.
